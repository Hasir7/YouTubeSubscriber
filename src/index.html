YouTube Subscribers API

A REST API built using Node.js, Express.js & MongoDB Atlas.

🟢 API Online

Available Endpoints

GET /subscribers
Returns all subscribers from the database.
GET /subscribers/names
Returns only subscriber names and subscribed channels.
GET /subscribers/:id
Returns subscriber details using the MongoDB Object ID.
